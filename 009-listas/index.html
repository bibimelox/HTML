Trabalhando com Listas

Listas Ordenadas

São listas que são numeradas, para isso usamos a tag ol.
Temos alguns tipos de configuração, para usá-las, usamos a tag type dentro do ol .

  1. arroz;
  2. feijão;
  3. batata;
  4. macarrão;
  5. salada.

Listas não ordenadas

São listas que não são numeradas, para isso usamos a tag ul.
Temos alguns tipos de configuração, para usá-las, usamos a tag type dentro do ul.

  • pão;
  • ovo;
  • manteiga;
  • presunto;
  • leite.

Mistura das listas
